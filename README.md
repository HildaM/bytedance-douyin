# bytedance-douyin

## 项目结构
├─api               存放接口
├─global            存放全局资源，database、nosql、logger...
├─initialize        存放初始化文件，初始化db、redis、cabin...
├─middleware        存放中间件，jwt认证、cabin权限...
├─model             存放实体类
├─router            存放路由配置
├─service           存放服务
└─utils             存放工具类
